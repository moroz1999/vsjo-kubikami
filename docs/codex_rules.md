# Codex Rules

1. Никогда не удалять пользовательские переносы строк.
2. Никогда не удалять пользовательские комментарии.
3. При правках в asm-файлах писать краткий комментарий справа от каждой добавленной или изменённой строки кода.
4. В `animate.a80` это правило применяется ко всем строкам кода, которые трогает Codex.
