# Route Map Viewer

Small browser viewer for room borders and enemy route points.
Wheel over the map zooms from `1x` to `4x`.

## Files

- `../../routes/manifest.json` is the route manifest used by the viewer and generator.
- `../../routes/room_X_Y.json` files store editable route points split by room.
- `routes-data.js` is generated by `generate_asm.php` from the split JSON data so `index.html` also works from a direct file open.
- `import_from_asm.php` imports current project route data from `routes/enemies.route*.a80`, `rooms.a80`, `rooms_unpacked/`, and route rewires in `logic/*.a80`.
- `generate_asm.php` generates `routes/enemies.route.a80`, `routes/enemies.route_X_Y.a80`, and `routes-data.js` from `routes/manifest.json` and `routes/room_X_Y.json`.

## Commands

```sh
php tools/map-viewer/import_from_asm.php
php tools/map-viewer/generate_asm.php
php tools/map-viewer/generate_asm.php --project
```

Generated asm files go to `routes/`. The `--project` option remains accepted for `compile.bat`.
